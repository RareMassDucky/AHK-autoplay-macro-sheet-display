# AHK-autoplay-macro-sheet-display

A small toolkit containing two independent AutoHotkey scripts designed for virtual piano software, including compatibility with Roblox.

### Tools Included
* **Autoplayer:** Automates the playing of virtual piano sheets.
* **Sheet Display & Tracker:** Assists in displaying and tracking sheet music to help you play along manually.

### Usage
1. Ensure [AutoHotkey](https://www.autohotkey.com/) is installed on your system.
2. Simply double-click the desired `.ahk` file to launch the script.
3. Each script operates independently; no complex configuration is required.
